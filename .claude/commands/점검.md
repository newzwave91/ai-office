지식금고 위키(vault/00_지식금고/)를 점검(lint)하라.

## 검사 항목

1. **모순**: 노트 간 서로 충돌하는 주장. 어느 쪽이 최신·정확인지 판단 근거와 함께 보고.
2. **중복**: 같은 사실을 다루는 노트가 분산되어 있는 경우 — 통합 후보 제시.
3. **고아 노트**: 어떤 노트의 [[위키링크]]로도 연결되지 않고 index.md에만 있는 노트.
4. **깨진 링크**: [[위키링크]]가 가리키는 노트가 실제로 존재하지 않는 경우.
5. **인덱스 정합성**: index.md 목록과 실제 파일 목록의 불일치(누락·유령 항목).
6. **낡은 정보**: 날짜가 오래됐거나 이후 노트·원문과 어긋나는 주장.
7. **빈 구멍**: 자주 언급되는데 자기 노트가 없는 개념, 채울 가치가 있는 데이터 공백 — 다음에 조사할 질문 제안.

## 처리 규칙

- 기계적으로 안전한 수정(인덱스 불일치 동기화, 깨진 링크 교정)은 바로 고친다.
- 내용 판단이 필요한 것(모순 해소, 노트 통합·삭제)은 고치지 말고 보고만 한다.
- 점검 결과를 vault/00_지식금고/log.md에 "## [YYYY-MM-DD] 점검 | 발견 N건" 으로 append한다.
- 보고 형식: 항목별 발견 사항 → 바로 고친 것 → 사용자 결정 필요한 것 → 제안 질문.
