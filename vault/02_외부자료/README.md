# 02_외부자료 — 외부 수신 자료 보관층

외부에서 다운받은 최신 자료를 모으는 곳. 01_원문과 동일한 불변(immutable) 원문층이다 — 여기 파일은 수정·삭제하지 않고, 핵심은 `/수집`으로 지식금고 노트에 증류한다.

## 분류 (예시 — 우리 사업에 맞게 자유롭게 만든다)
- 사업·주제별 하위 폴더를 만들어 자료를 모은다 (예: `트렌드/`, `경쟁사/`, `제품/`).
- **github/** — 레포 README, 관심 코드 조각, 릴리스 노트. 레포 전체 클론은 넣지 말 것(볼트가 비대해짐) — 필요한 파일이나 정리본만.
- **claude-code/** — 클로드코드 팁, 개발자 가이드, 패턴 문서.

## 사용법
1. 자료를 해당 분류 폴더에 넣는다 (파일명: "YYYY-MM-DD 제목.md" 권장)
2. `/수집 vault/02_외부자료/분류/파일명` 으로 지식금고에 통합
3. 이후 작업에서 팀장들은 증류된 노트를 읽고, 필요할 때만 여기 원문을 참조한다
